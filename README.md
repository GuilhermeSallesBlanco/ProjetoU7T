# Descrição do projeto
Trata-se de um dispositivo detector de campos eletromagnéticos baseado na Raspberry Pi Pico W, integrado à BitDogLab. O sistema utiliza uma antena telescópica como sensor para detectar variações no campo eletromagnético, como as geradas por fontes de energia elétrica. Os sinais captados são processados pelo microcontrolador RP2040 por meio de sua entrada Conversora Analógica-Digital (Analog-Digital Converter – ADC), que gera alertas sonoros e visuais de acordo com os valores recebidos.
